# ENVS-193DS_homework-03

## General information

This repository contains my work for ENVS 193DS Homework 03. The assignment includes statistical analysis, data visualization, personal data exploration, an affective visualization draft, and a statistical critique of a research paper.

## Data and file information

The `data` folder contains the datasets used in this assignment.

- `temp-kelp.csv`: Data on ocean temperature and giant kelp frond elongation rate.
- `coffee_data 2.csv`: My personal daily habits dataset, including sleep, coffee consumption, work hours, schoolwork hours, wake time, and exercise.

The main homework file is:

- `ENVS-193DS_homework-03.qmd`: Quarto file containing the code, written responses, figures, and images for the homework.

The rendered output file is:

- `ENVS-193DS_homework-03.html`: Rendered version of the homework.
